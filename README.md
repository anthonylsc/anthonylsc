# Anthony Portfolio

A small React + Vite portfolio site.

## Requirements
- Node.js 16+

## Install
Double-click `install.bat` or run in PowerShell:

```powershell
cd path\to\anthony.lsc
install.bat
```

## Run
Double-click `run-dev.bat` or:

```powershell
cd path\to\anthony.lsc
run-dev.bat
```

Open `http://localhost:5174` in your browser.
